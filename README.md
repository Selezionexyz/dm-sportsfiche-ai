# DM Sports AI Generator

Générateur de fiches produit optimisé pour les marques sportswear.

## Features
- Interface style DM Sports
- 28 marques sportswear
- Export multi-format
- Optimisation SEO

## Installation
1. Clone le repo
2. `npm install`
3. `npm start`

## Usage
Accédez à http://localhost:3000
